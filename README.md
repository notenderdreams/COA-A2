# COA-A2 - Finite State Machine Simulator

## Getting Started

### Prerequisites

This project uses [Bun](https://bun.sh/) for package management and as a runtime, but you can also use [Node.js](https://nodejs.org/) with npm.

- **Bun** (Recommended)
  ```bash
  curl -fsSL https://bun.sh/install | bash
  ```
- OR **Node.js** (v20+)

### Local Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/notenderdreams/COA-A2.git
   cd COA-A2
   ```

2. **Install Dependencies**

   ```bash
   # Using Bun
   bun install

   # OR using npm
   npm install
   ```

3. **Run Development Server**

   ```bash
   # Using Bun
   bun run dev

   # OR using npm
   npm run dev
   ```

4. **Visit the App**
   Open [http://localhost:5173](http://localhost:5173) in your browser.
